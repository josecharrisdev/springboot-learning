# Spring Boot Learning Path

Este repositorio contiene múltiples proyectos desarrollados como parte de mi proceso de aprendizaje en Spring Boot.

La ruta de estudio fue diseñada con el apoyo de ChatGPT, y mi aprendizaje se limitará exclusivamente a información dada por IA durante el proceso. He decidido 
documentar cada etapa mediante proyectos prácticos con el objetivo de evidenciar mi progreso, y experimentar el nivel al que se puede llegar con el uso exclusivo
de IA en la formación

### 1️⃣ Conceptos básicos
- IoC: Inversión de control
- DI: Inyección de dependencias
